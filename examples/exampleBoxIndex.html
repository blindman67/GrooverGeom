A Box represents a bounds and always has its edges aligned to the x and y axis. It is defined by 4 numbers that represent the top, left, right, and bottom. A box can be obtained from all primitives that can be represented via a box by calling primitive.asBox([box]). The optional argument box is supplied will extend the bounds of the passed box to include the primitive.
